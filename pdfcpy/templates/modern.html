Tesseract OCR

Installation:

  • Download from Tesseract OCR GitHub
  • Choose Windows installer (not source code)
  • Install to default location
  • Path should be: C:\Program Files\Tesseract-OCR\tesseract.exe

LibreOffice

Current Setup:

  • LibreOffice is installed in: ./assets/Libre/
  • Executable path: ./assets/Libre/program/soffice.exe
  • This should work automatically for document conversions

Ghostscript

Installation:

  • Download from Ghostscript
  • Install for Windows (64-bit recommended)
  • Find gswin64c.exe in installation directory
